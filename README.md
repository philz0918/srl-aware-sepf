# SRL-Aware Analysis

This repository contains code for **SRL-aware Sentiment, Emotion, Politeness, and formality analysis**.

## Overview

The goal of this project is to use **Semantic Role Labeling (SRL)** to improve language understanding by incorporating predicate–argument structure and semantic roles into downstream analysis tasks.  
This repository focuses on four tasks:

- **Sentiment detection**
- **Emotion detection**
- **Politeness detection**
- **Formality detection**


## Repository Contents

- `srl_aware_sentiment/` — SRL-aware sentiment detection code
- `srl_aware_emotion/` — SRL-aware emotion detection code
- `srl_aware_politeness/` — SRL-aware politeness detection code
- `srl_aware_formality/` — SRL-aware formality detection code

- `README.md` — project description

## Features

- Text preprocessing pipeline
- SRL-based feature extraction
- Task-specific analysis for sentiment, emotion, politeness, and formality
- Example scripts for running the models
